# Mesociclo 01 — Adaptacao ao PPL + Recomposicao

Parte do [[central]]

## Objetivo

Transicao do bro split (1x/semana por musculo) para PPL com frequencia 2x/semana, em deficit calorico moderado para recomposicao corporal. Foco em emagrecimento com preservacao muscular.

## Estrutura

| Item | Valor |
|------|-------|
| Duracao | 4 semanas (3 progressivas + 1 deload) |
| Split | PPL (Push/Pull/Legs) 6x/semana |
| Periodizacao | DUP (Pesado → Moderado → Metabolico) |
| Volume | 14-16 sets/musculo/semana |
| Cardio | Escada 3x/semana, 25 min, speed 10 (seg/qua/sex) |
| Abdominal | 3x/semana ao final do treino (ter/qui/sab) |
| Rest | Domingo |
| Deload | Semana 4 (volume -50%, intensidade mantida) |

## Split Semanal

| Dia | Treino | Intensidade |
|-----|--------|-------------|
| Segunda | Push A (Peito/Ombro/Triceps) + Escada | Pesado (6-8 reps, RPE 8-9) |
| Terca | Pull A (Costas/Biceps/Trapezio) + Abdominal | Pesado (6-8 reps, RPE 8-9) |
| Quarta | Legs A (Quad/Posterior/Panturrilha) + Escada | Pesado (6-8 reps, RPE 8-9) |
| Quinta | Push B (Peito/Ombro/Triceps) + Abdominal | Moderado (10-12 reps, RPE 7-8) |
| Sexta | Pull B (Costas/Biceps/Trapezio) + Escada | Moderado (10-12 reps, RPE 7-8) |
| Sabado | Legs B (Quad/Posterior/Panturrilha) + Abdominal | Metabolico (12-15 reps, RPE 7) |
| Domingo | REST | Descanso |

## Progressao

- Semana 1: Cargas de referencia (RPE 8-9 no pesado, 7-8 no moderado)
- Semana 2: +2.5kg em compostos, +1 rep em isoladores
- Semana 3: +2.5kg em compostos (ou +1 rep), manter isoladores
- Semana 4: DELOAD — mesmos exercicios, 2 series (em vez de 3-4), mesma carga
